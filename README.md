# Digital Second Brain (Web Clipper)

A personal archiving assistant developed to prevent information clutter on the internet and avoid losing valuable content (text, links, images).

## Features

- **Chrome Extension:** Quick saving with right-click or selection.
- **Smart Screenshot:** Select any area on the page and save it as a high-quality image.
- **Masonry Layout:** Modern and dynamic interface similar to Pinterest.
- **Cinema Mode (Lightbox):** View saved images in full screen.
- **Security:** JWT-based Login/Registration system.

## Technologies Used

- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Frontend:** HTML5, CSS3, Vanilla JS
- **Extension:** Chrome Manifest V3
